Welcome to the PHYTEC BSP-Yocto buildsystem for the phyCORE-STM32MP SOM.

Before building for the first time, you should take a look in the main
configuration file 'conf/local.conf'. There you can tweak the build system
settings and license configuration.

Default build target for this build configuration are:

      st-image-weston       -   OpenSTLinux weston image with basic Wayland support (require 'openstlinux-weston' distro)
      st-image-core         -   OpenSTLinux core image (compatible with all OpenSTLinux distro except with 'openstlinux-tiny')

Other available images proposed as example only:
      st-image-core-minimal       -   PHYTEC OpenSTLinux core minimal image (require 'openstlinux-tiny' distro)
      st-example-image-qt         -   OpenSTLinux example of image based on QT framework   (require 'openstlinux-eglfs' distro)
      st-example-image-qt-minimal -   PHYTEC OpenSTLinux example of image based on QT minimal framework (require 'openstlinux-eglfs' distro)
      st-example-image-x11        -   OpenSTLinux example of image based on X11            (require 'openstlinux-x11' distro)
      st-example-image-xfce       -   OpenSTLinux example of image based on XFCE framework (require 'openstlinux-x11' distro)
